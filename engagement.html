Explain the difference between inline, embedded, and external CSS.

Inline CSS is applied directly to an element, embedded CSS is placed within style in the HTML head, and external CSS is stored in a separate file and linked. External CSS is best because it keeps code clean, reusable, and easier to maintain. A selector targets HTML elements for styling, like p { color: blue; } to style all p tags.

Explain why you should only use external CSS.

External CSS keeps styles separate from HTML, making code cleaner, reusable, and easier to maintain across multiple pages.

What is a selector used for?

A selector is used to target HTML elements for styling. Example: p { color: blue; } selects all p elements and applies blue text.
